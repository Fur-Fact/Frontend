# Fur-fact-Frontend

## Branch Naming Convention

`이슈 태그(ex. chore, feat, fix)`+`/`+`이슈 번호`+`짧은 설명(영문)` \
ex. `feat/4-make-login-page`

## 폴더명, 파일명 Convention

폴더명: `snake-case` 사용

- ex) `loginPage`, `userAuth` 등

파일명: vue: `PascalCase` 통일, 함수 `camelCase`

## .gitmessage 적용법

```bash
git config commit.template .gitmessage.txt
```
